# Host

## Definition

A host is a device connected to a network that has its own IP address and can send, receive, or process data over the network.


## Characteristics

* Has a unique IP address within a network.
* Can send and receive data.
* Can act as a client, a server, or both depending on the communication.
* Can communicate with other hosts on the same network or across the Internet.


## Examples

* Laptop
* Desktop Computer
* Smartphone
* Server
* Virtual Machine (VM)
* Network Printer


## Important Points

* Every server is a host.
* Not every host is a server.
* A host does not need to be connected to the Internet; it only needs to be connected to a network.
* The role of a host can change depending on the communication.


## Interview Notes

**Q: What is a host?**
A: A host is a device connected to a network that has an IP address and can send, receive, or process data.

**Q: Is every host a server?**
A: No. Every server is a host, but not every host is a server.

**Q: Can a host act as both a client and a server?**
A: Yes. Depending on the communication, a host can act as a client, a server, or both.

# Network

## Definition

A network is a collection of interconnected devices that can communicate and exchange data with each other.


## Characteristics

* Consists of multiple connected devices (hosts).
* Enables communication and data sharing between devices.
* Can be wired or wireless.
* Can be small (home network) or large (corporate network).
* Uses protocols to facilitate communication between devices.


## Examples

* Home Wi-Fi Network
* College Campus Network
* Office Network
* Mobile Carrier Network
* Data Center Network


## Important Points

* A network allows hosts to communicate with each other.
* Devices in a network can share resources such as files, printers, and Internet access.
* Networks can be connected together to form larger networks.
* The Internet is the largest example of interconnected networks.


## Interview Notes

**Q: What is a network?**
A: A network is a collection of interconnected devices that can communicate and exchange data with each other.

**Q: What is the purpose of a network?**
A: The purpose of a network is to enable communication, data exchange, and resource sharing between devices.

**Q: What is the difference between a network and the Internet?**
A: A network is a group of connected devices, while the Internet is a global collection of interconnected networks.

# CLIENT

## Definition
A client is a device/application connected to the network that initiates a conversation and requests for data or servicies.

## Examples
- A browser requesting a webpage
- AWS CLI requesting for AWS services
- SSH client connecting to a server

## Important points
- A client initiates conversation
- A client requests for services
- A client can be a device or an application


# CLIENT-SERVER MODEL

## Definition 

A client-server model is a communication architecture in which a client requests data or services to a server and the server processes the request and provide the data or service to the client


## Flow

Client -> Request -> Server -> Response -> Client


## Example

1. A user enters "youtube.com" in a browser
2. Browser sends a request to YouTube's server.
3. The server processes the request.
4. The server sends back the webpage to the browser.

In this interaction:
1. Browser = Client
2. YouTube infrastructure = Server

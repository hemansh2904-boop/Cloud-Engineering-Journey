# Server

## Definition

A server is a device, application, or service that provides data, resources, or functionality to clients over a network.


## Characteristics

* Provides services or resources to clients.
* Waits for and responds to client requests.
* Can serve multiple clients simultaneously.
* Has an IP address and is therefore a host.
* Can be physical, virtual, or cloud-based.


## Examples

* Web Server (YouTube, Google)
* DNS Server
* Database Server
* File Server
* SSH Server
* AWS EC2 Instance hosting a website


## Important Points

* Every server is a host.
* Not every host is a server.
* A server provides services, while a client requests them.
* A server can be hardware, software, or both.
* A server may serve many clients at the same time.


## Interview Notes

**Q: What is a server?**
A: A server is a device, application, or service that provides data, resources, or functionality to clients over a network.

**Q: Is every server a host?**
A: Yes. Every server is a host because it is connected to a network and has an IP address.

**Q: What is the difference between a client and a server?**
A: A client initiates communication and requests services, while a server processes those requests and provides the requested services or data.

**Q: Can a server also act as a client?**
A: Yes. Depending on the communication, a device can act as both a server and a client.
